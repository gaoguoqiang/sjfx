献给即将创业或有创业想法的你！

文 / 公司宝典
2016-04-20 20:13

升学、恋爱、成家、创业、离别，生命中总有那么一段时光，充满恐惧，可是除了勇敢面对，我们别无选择。

一庥&JerryZ 作品


找人 -> 打杂 -> 找钱 -> 找人 -> 打杂 -> 找钱 -> 找人 -> 打杂 -> 找钱 -> 找人 ……

看上去，This is an endless loop.

这就是很多创业者的真实创业生活。创业者是怎么走进这个死循环的？他们又应该怎样走出这个死循环？

创业者的自嗨想法

1.将创业过于理想化，他们容易将自己的创业项目的发展置于一个理想化的发展过程。但往往现实让他们措手不及。

2.一定程度上认为自己的创业项目几乎没有竞争对手，只要我做出来就能占领市场。

3. 集权观念太重，事无巨细，芝麻大的事情都要亲力亲为，恨不得自己亲自去写产品文档，去码代码，所有人的所有决定必须经由自己拍板才能实施。

4. 过多的去看“行业战略”，过多的去谈“用户体验”，忽略第一需求就是“先让自己活下来”

3. 过多干涉下属的工作，做不到“让专业的人去做专业的事”

5. 过分依赖外部“资源”，可以拿来说给投资人听，可以拿来给应聘者听，有时候把自己都骗过去了。如果这种“资源”是核心竞争力的话，BAT是不是分分钟就可以把你秒掉了？

6. 做规划的时候喜欢用一句话带过那些实际上很难解决的问题，做起来的时候就傻眼了。

7. 拍大腿定时间排期，产品东一下，项目西一下。一年365天，365个想法，380个目标，搞哪样？玩打地鼠啊？结果是一个又一个时间点被miss掉！

8.自身idea很好，执行力不强，导致团队效率低下不够果决。

员工眼中创业者的问题

1.战线过长，公司一共没几个人，做做这，做做那，感觉哪哪都可能成功，其实哪哪都是狗屎；

2.明确知道公司中层有搅屎棍，内耗严重，还是不管不顾，有失偏颇；

3.对员工小承诺小奖励记住不，时发时不发；大承诺玩文字游戏，耍人耍几年；

4.喜欢看员工社交平台表忠心，晒用功；

5.会哭的孩子有奶喝；

6.总有种智商上的优越感……

7.还没什么蛋糕就天天怕别人惦记着他，鄙帚自珍；

8.结果人才放不开，能干事的人才都被逼走，留下一堆表忠心的搅屎棍。

给创业者的一些忠告

产品

设计好产品，越傻瓜越好。

清楚自己的产品定位和核心竞争优势，以此为核心开展公司各项事务！

保持专注，做你最主要的事情。不断思考自己在解决什么需求，以及用什么方法将此需求满足至极致。

互联网的核心是垄断而非竞争，创业公司应该先去找到一个细分市场并垄断之，然后再逐步扩大。

最好创业的同时就能构建好商业模式。

融资

好的投资人都是你感召来的，是他们来找你，不是你去找他们。忘记融资的事情，找到一位愿意付费的顾客更加重要，然后去找第二位，第三位...有了顾客之后，投资者自然会找上门来。这才是融资的最高境界。

团队

决定创业公司的命运，很多时候取决于创始人一个人。更多时候，取决于团队。一万零一个失败的创业故事，至少有一万个是因为团队而失败的。找到适合的合伙人并不比找一个伴侣容易！

对团队的激励，一定要规则清晰、透明、可预期。不论是期权还是薪酬。

招聘

大部分事情不用亲力亲为。招人一定要亲力亲为。记住最棒的公司总是和最棒的人才相辅相成的。要雇佣到优秀的人才，你必须画大饼。除了希望在一个优秀的团队工作之外，候选人更需要看到你画的大饼。

当你找到了合适的员工后，留住人才至关重要！一个善意的眼神、几次团建活动，一个惊喜的生日礼物，经常和员工共同你的想法和了解员工的想法，花不了你多少钱和时间，却能让员工感受的关爱，更努力的投入工作中来

其他

一定保管好公章，财务章，私人印章，以及营业执照，组织机构代码证，银行票单，三联单等相关资料，千万不要再要签合同时才发现找不到啦。
